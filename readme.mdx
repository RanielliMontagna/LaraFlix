# LaraFlix

Projeto criado para aprimorar o conhecimento em desenvolvimento de software, com foco em PHP e Laravel.

## Sobre o projeto

O projeto consiste em cadastro de atores e filmes, onde o usuário pode
cadastrar atores e filmes, e relacionar os atores aos filmes.

Até o momento o projeto conta com as seguintes funcionalidades:

-   [x] Sistema de login e registro de usuários
-   [x] Cadastro de atores
-   [ ] Cadastro de filmes

## Tecnologias utilizadas

-   [PHP](https://www.php.net/) - Linguagem de programação
-   [Laravel](https://laravel.com/) - Framework PHP para desenvolvimento web
-   [AdminLTE](https://adminlte.io/) - Template Admin para facilitar a criação de telas de administração e painéis de controle web

## Instalação

-   Clone o repositório
-   Instale as dependências do projeto com o comando `composer install`
-   Copie o arquivo `.env.example` para `.env` e configure as variáveis de ambiente
-   Gere a chave da aplicação com o comando `php artisan key:generate`
-   Execute as migrations com o comando `php artisan migrate`
-   Execute o comando `php artisan serve` para iniciar o servidor de desenvolvimento
